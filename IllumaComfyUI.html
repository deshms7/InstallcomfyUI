Basic Usage
Run your first image generation in a few clicks.

Generating an image

Select a model — in the Load Checkpoint node, pick a .safetensors file from the dropdown.
Enter your prompt — click the positive CLIP Text Encode node and type what you want to generate.
Set image size — in the Empty Latent Image node. Use 512×512 for SD1.5, 1024×1024 for SDXL.
Click Queue Prompt — the blue button, top right. Generation starts immediately on the GPU.
View your image — it appears in the Save Image node and is saved automatically to the outputs folder.

File Locations
All data lives outside the container and persists across restarts.
PathContents
/data/comfyui/models/checkpoints/Main model files (.safetensors, .ckpt)
/data/comfyui/models/loras/LoRA / LyCORIS weights
/data/comfyui/models/vae/VAE decoder files
/data/comfyui/models/controlnet/ControlNet models
/data/comfyui/outputs/Generated images, organised by date
/data/comfyui/custom_nodes/ComfyUI extensions
ℹ️ Files placed in these paths are immediately visible to ComfyUI — no restart needed for models.
Adding Models
Download models from HuggingFace or CivitAI directly onto the machine.

Download via terminal

Open a terminal — right-click the desktop → Open Terminal.
Run wget with your model URL: 
sudo wget -c -O /data/comfyui/models/checkpoints/model-name.safetensors \
  https://huggingface.co/<author>/<repo>/resolve/main/model.safetensors
Refresh the dropdown — click Refresh in the Load Checkpoint node in ComfyUI. Your model appears immediately.

Managing the Service
ComfyUI runs as a systemd service and starts automatically on boot.
CommandWhat it does
sudo systemctl status comfyuiCheck if running
sudo systemctl restart comfyuiRestart the service
sudo systemctl stop comfyuiStop the service
sudo journalctl -u comfyui -fLive logs
sudo docker logs comfyui --tail 50Last 50 lines of container output
sudo docker exec comfyui nvidia-smiCheck GPU inside container
Finding Outputs
Every generated image is saved automatically.

Output folder

/data/comfyui/outputs/

Images are organised into subfolders by date. Open the file manager on the desktop and navigate there, or use the terminal:

ls -lt /data/comfyui/outputs/ | head -20
💡 The outputs folder is outside the container — images are safe even if the container is updated or recreated.
